# 🎸 Грим та Грім — Landing Page

Сучасний адаптивний лендинг для українського рок-гурту **«Грим та Грім»**.
Проєкт демонструє афішу концертів, інформацію про гурт та форму для зв'язку з інтеграцією Telegram.

## 📌 Основний функціонал

* 🎤 **Hero-секція**

  * Презентація гурту
  * Кнопка "Замовити квиток" зі скролом до концертів

* 📅 **Найближчі концерти**

  * Список подій
  * Кнопки відкривають **модальне вікно з формою** бронювання

* 👥 **Учасники гурту**

  * Статичний блок з картками учасників

* 📖 **Наша історія**

  * Текстовий опис гурту

* 📩 **Форма зв'язку**

  * Відправка повідомлення через **Telegram Bot**
  * Валідація полів
  * Нотифікації через `iziToast`

* 🗺️ **Карта**

  * Вбудована через iframe

## 🔔 Поведінка інтерфейсу

* Клік по **"Замовити квиток"** → плавний скрол до секції концертів
* Клік по кнопці в концертах → відкривається **модальне вікно**
* Форма:

  * перевіряє введені дані
  * показує статус через toast-повідомлення

 * Контактна форма:

  * перевіряє введені дані
  * надсилає повідомлення в Telegram
  * показує статус через toast-повідомлення
